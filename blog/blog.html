what is the document object modal (DOM)

The Document Object Model (DOM) is a programming interface that allows you to manipulate the structure and content of a web page using JavaScript and other programming languages. In simpler terms, the DOM represents an HTML or XML document as a tree structure, where each element or tag in the document is represented as a node. This structure allows developers to easily access and manipulate the content, style, and behavior of web pages dynamically.

How do you select on element from DOM?

An element from the DOM, JavaScript provides several powerful methods designed to locate specific elements or groups of elements based on their attributes, tag names, or other characteristics. Selecting elements from the DOM is a fundamental task for manipulating web pages dynamically, allowing developers to update the content, apply styles, or respond to user interactions.

What is event delegation in the context of the DOM, and why is it useful?

The DOM follows a process called event propagation, where events triggered on an element "bubble up" through the parent elements, all the way to the root of the document (the document object). Event delegation takes advantage of this by adding a single event listener to a parent element. When an event (such as a click) is triggered by a child element, the event bubbles up to the parent, where the listener is set.

How do you manipulate and element attributes and styles using the DOM?

Manipulating element attributes and styles in the DOM using JavaScript allows developers to dynamically change how elements behave and appear on a webpage. This is essential for creating interactive and dynamic web pages.
